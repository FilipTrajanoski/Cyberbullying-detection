# Cyberbullying detection
This project trains BERT and RoBERTa models to detect cyberbullying in text.
## Datasets
The following datasets are being used in the project:
1. https://www.kaggle.com/datasets/soorajtomar/cyberbullying-tweets
2. https://github.com/mohamadahj/Cyberbully-Detection-Dataset/blob/main/cb_multi_labeled_balanced.csv
3. https://www.kaggle.com/datasets/waalbannyantudre/hate-speech-detection-curated-dataset?select=HateSpeechDatasetBalanced.csv
4. https://www.kaggle.com/code/andrewmvd/cyberbullying-classification-getting-started/input?select=cyberbullying_tweets.csv
